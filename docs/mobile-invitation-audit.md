# Mobile Invitation Feature Audit

## Reference Signals

최근 모바일 청첩장/웨딩 QR 서비스에서 반복적으로 강조되는 기능은 다음과 같습니다.

- RSVP 실시간 집계와 엑셀/CSV 내보내기
- 신랑측/신부측 하객 분리 집계
- 식사 여부, 메모, 연락처 수집
- QR 코드 기반 공유와 현장 체크인
- 방명록 승인/숨김 관리
- 하객 사진/영상 포토드롭
- D-day, 지도, 길찾기, 계좌 복사, 카카오 공유

## Current Coverage

MC Page는 현재 모바일 청첩장 편집, 공개 URL, 지도, 갤러리, 계좌 복사, RSVP 제출, 방명록, QR/공유, 식전영상 제작 흐름을 갖추고 있습니다.

## Gap Closed In This Pass

하객 응답 관리가 부족했습니다. 공개 페이지에서 RSVP와 방명록을 받을 수는 있었지만, 제작자가 실제로 응답을 집계하고 정리하는 운영 화면이 약했습니다.

이번 작업으로 청첩장 수정 화면에 `하객 응답 관리` 섹션을 추가했습니다.

- 참석 응답 수, 참석 예정 인원, 불참/미정, 방명록 상태 요약
- 신랑측/신부측/양측/미선택 인원 분리 집계
- 최근 RSVP 응답 테이블
- CSV 다운로드
- 방명록 공개/숨김 관리

## Next Recommended Features

- QR 체크인: 예식 당일 하객 방문 확인
- 포토드롭: 하객이 찍은 사진/영상을 QR로 업로드
- RSVP 질문 커스터마이징: 성인/어린이/유아, 식사 여부, 버스 탑승 등
- 카카오 알림/문자 리마인더: 미응답자에게 일정 전 알림
- 방명록 승인 모드: 작성 즉시 공개 대신 검토 후 공개
